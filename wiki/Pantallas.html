Pantallas

Las pantallas son menus, por ejemplo la pantalla principal o la de opciones también.

Crear una pantalla

Para crear un objeto es tan fácil como poner su nombre. 

		
			local main = {}

			function main.load(modCreator, api) 
                            modCreatior.newScreen("testScreen")
			end
			
			return main
