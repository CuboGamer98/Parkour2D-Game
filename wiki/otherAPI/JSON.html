JSON

Esta es una API para pasar tablas a texto y viceversa.

Como usar la API

En la API solo existen dos funciones importantes: api.JSON:encode(<tabla>) y api.JSON:decode(<texto>).

api.JSON:encode(<tabla>): Cuando le proporcionas una tabla, te la devuelve en forma de texto.
api.JSON:decode(<texto>): Cuando le proporcionas una table en forma de texto, te devuelve la tabla.

Ejemplo: 

		
			local main = {}

			function main.load(modCreator, api) 
                            local table = {}
                            table.coins = 30
                            table.emeralds = 10

                            local string = api.JSON:encode(table)
                            print(string) --> {coins = 30,emeralds = 10}

                            local newTable = api.JSON:decode(string)
                            print(newTable) --> table 0xdsfguhj
			end
			
			return main
