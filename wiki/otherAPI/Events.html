Events

Es una API que proporcina eventos, que significa esto, que podeis crear eventos y recibir informacion de eventos que han sucedido.

Usar eventos del juego

Di os fijasteis bien, en la funcion main.conf() hay un apartado que dice conf.Events = {}, en esa funcion tendreis que habilitar todos los eventos que vallais a usar. 

		
			function main.conf()
                            local conf = {}
			    conf.JSON = false
			    conf.screen = false
			    conf.modList = false
	
                            conf.Events = {}
                            conf.Events.enabled = true --> Importante
                            conf.Events.plrJump = false
                            conf.Events.plrWalk = false
                            conf.Events.plrGetDamage = false
	
                            conf.Events.click_modMode = false
                            conf.Events.click_FullScreen = false
                            conf.Events.click_language = false
                            conf.Events.click_controls = false
                            conf.Events.click_options_back = false
                            conf.Events.click_controls_back = false
                            conf.Events.click_laguaje_es_es = false
                            conf.Events.click_laguaje_en_us = false
                            conf.Events.click_languaje_back = false
                            conf.Events.click_start = false
                            conf.Events.click_options = false
                            conf.Events.click_quit = false

			    return conf
                        end

Crear evento

Para crear un evento solo tendreis que usar esta linea de código:

api.Events:newEvent(<nombre del evento>)

Ejemplo: 

		
			local main = {}

			function main.load(modCreator, api) 
                            api.Events:newEvent("coin_collected")
			end
			
			return main

Ejecutar evento

api.Events:fire(<nombre del evento>) 

		
			local main = {}

			function main.load(modCreator, api) 
                            api.Events:fire("coin_collected")
			end
			
			return main

Ejecutar funcion cuando evento suceda

api.Events:onFire(<nombre del evento>, <función>) 

		
			local main = {}

			function main.load(modCreator, api) 
                            api.Events:onFire("coin_collected", function()
                                print("Funciono!")
                            end)
			end
			
			return main
