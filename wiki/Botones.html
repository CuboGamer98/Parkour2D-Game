Wiki de PARKOUR2D

Botones

Los botones son bloques 2D que cuando se interactua en ellos se ejecuta una acción.

Crear boton en una pantalla

Para crear un boton es solamente rellenar la tabla que teneis abajo.

modCreatior.newButtonFES( <nombre de la pantalla>, {
text = <texto que quieres que valla en el boton>,
x = <posicion en horizontal>,
y = <posicion en vertical>,
width = <anchura del boton>,
height = <altura del boton>,
fontM = <numero para cambiar el tamaño del texto> (opcional)
})

Ejemplo: 

		
			local main = {}

			function main.load(modCreator, api) 
                            modCreatior.newButtonFES("testScreen", {
                                text = "test_button",
                                x = 100,
                                y = 100,
                                width = 300,
                                height = 100,
                                fontM = 1
                            })
			end
			
			return main
