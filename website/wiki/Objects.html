Wiki de PARKOUR2D

Objetos

Los objetos son bloques con caracteristicas únicas, por ejemplo los pinchos clavan y hacen daño, con lo que es objecto, pero el suelo no es un objeto ya que no tiene utilidad alguna.

Crear objeto del juego

Para crear un objeto es tan fácil como poner su nombre y las coordenadas X y Y 

		
			local main = {}

			function main.load(modCreator, api) 
                            modCreatior.newObject("coin", {
                                x = 100,
                                y = 100
                            })
			end
			
			return main

En ese trozo de código lo que estamos haciendo es crear una moneda en la posicion 100,100 (x,y).

Lista de objetos

- "spikes": Pinchos que hacen daño.

- "coin": Moneda que da puntos.

- "box": Caja con físicas.

- "red_slime_entity": Enemigo que solo salta pero hace daño.

- "platform": Plataforma en la que si apretas shift caes.
