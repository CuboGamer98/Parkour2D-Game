screen

Esta API proporciona varias funciones para poder manejar la pantalla principal de forma rápida.

Como usar la API

En la API solo existen dos funciones importantes:

api.screen.setVisibilityN(<nombre de la pantalla>, <estado>), api.screen.getVisibility(<nombre de la pantalla>) y
api.screen.changeButtonPos(<nombre del boton>, <x>, <y>).

api.screen.setVisibilityN(<nombre de la pantalla>, <estado>): Si colocas el nombre de la pantalla y su estado, le cambias la visibilidad (true = visible, false = invisible).
api.screen.getVisibility(<nombre de la pantalla>): Si le proporcionas el nombre de una pantalla, te devolvera su visibilidad.
api.screen.changeButtonPos(<nombre del boton>, <x>, <y>): Cuando le proporcionas el nombre de un boton y unas coordenadas, el boton cambiara a la nueva posición.

Ejemplo: 

		
			local main = {}

			function main.load(modCreator, api) 
                            local mainVisibility = api.screen.getVisibility("main")
                            if mainVisibility then
                                api.screen.setVisibilityN("main", false)
                            end
			end
			
			return main

Lo que estoy haciendo en este trozo de código es que si la visibilidad de la pantalla principal es true entonces pongo la pantalla invisible
