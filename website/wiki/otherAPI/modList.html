modList

modList es una función que cuando la ejecutas te devuelve una tabla con todos los mods cargados en el juego ahora mismo.

Usar función

Ejemplo: 

		
			local main = {}

			function main.load(modCreator, api) 
                            local mods = api.getModList()
			end
			
			return main

Cuando la función se ejecuto le dio a la variable este valor: 

		
        {
            { 
                name = "mod_menu",
                modMainPath = "C:\\Users\\CuboGamer98\\AppData\\Roaming\\Parkour2D\\Mods\\mod_menu",
                modMainModule = "C:\\Users\\CuboGamer98\\AppData\\Roaming\\Parkour2D\\Mods\\mod_menu\\main.lua", 
                module = { 
                    conf = <function 1>, 
                    load = <function 2>
                }
            }
        }
