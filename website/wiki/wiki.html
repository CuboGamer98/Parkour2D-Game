1 – Introducción

Parkour2D es un juego plataforma básico preparado para que los jugadores puedan crear juegos sin ningún problema.

Esta wiki proporciona a los jugadores novatos en programación para ayudarles a que ellos también puedan crear sus propios mods.

Es recomendable para la creación de mods saber Lua

2 – Como empezar

Para comenzar con la creación de mods hay que crear una carpeta donde estarán todos los archivos, para ello habrá que ir a %appdata% u localizar la carpeta Parkour2D.

Una vez localizada la carpeta procedemos a abrirla. Estando aquí si no está la carpeta Mods habría que crear una, pero si existe ya solo tienes que abrirla y ahora habría que crear la carpeta donde irán todos los archivos del mod, es obligatorio que sea el nombre del mod y que no tenga espacios.

Después de haber creado la carpeta de tu mod procedes a abrirla y dentro tienes que crear un archivo llamado main.lua. Debería de quedar algo tal que así:

		
            AppData/Roaming/Parkour2D/Mods/<tu mod>/main.lua

Estos pasos son los más importantes para que el mod funcione correctamente, tenéis que abrir el archivo y colocar este código: 

		
			local main = {}

			function main.load(modCreator, api)

			end
			
			return main

Hay un apartado opcional que podéis colocar si vais a usar la segunda API que es main.config(), en esta podeis elegir que vais a usar cambiando false por true.

		
			local main = {}

			function main.conf()
                            local conf = {}
			    conf.JSON = false
			    conf.screen = false
			    conf.modList = false
	
                            conf.Events = {}
                            conf.Events.enabled = false
                            conf.Events.plrJump = false
                            conf.Events.plrWalk = false
                            conf.Events.plrGetDamage = false
	
                            conf.Events.click_modMode = false
                            conf.Events.click_FullScreen = false
                            conf.Events.click_language = false
                            conf.Events.click_controls = false
                            conf.Events.click_options_back = false
                            conf.Events.click_controls_back = false
                            conf.Events.click_laguaje_es_es = false
                            conf.Events.click_laguaje_en_us = false
                            conf.Events.click_languaje_back = false
                            conf.Events.click_start = false
                            conf.Events.click_options = false
                            conf.Events.click_quit = false

			    return conf
                        end

			function main.load(modCreator, api)

			end
			
			return main

Teniendo todo esto hecho podéis empezar a programar el mod.

3 – Como empezar a programar el mod

Para empezar debes saber que todo el código siempre va a empezar a ejecutarse desde la función main.load(). Que significa esto, que todo lo que pongáis en la función main.load() se ejecutara nada más empezar el juego.

Si os disteis cuenta anteriormente, en la función main.load() había dos argumentos: modCreator y api, los dos argumentos son APIs que ofrece el mod para que se os haga más fácil la creación de mods.

Si teneis alguna duda de algo ir al servidor de Discord de Parkour2D y preguntar.
